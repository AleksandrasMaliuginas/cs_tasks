# Namų darbas

## 1 užduotis - Bėgikai

Duomenų faile `duomenys.txt` pirmasis skaičius nurodo, kiek bėgikų dalyvavo maratone. Toliau pateikti bėgikų bėgimo laikai minutėmis (gali būti su kableliu). Atspausdinkite visus bėgimo laikus, kurie buvo greitesni nei 4 valandos.

**Reikalavimai**:
- 4 valandos = 240 minučių.
- Išveskite tik tuos laikus, kurie yra mažesni už 240.
- Kiekvieną tinkamą laiką spausdinkite atskiroje eilutėje.

Įvestis (`duomenys.txt`):
```
6
255
239.5
180
240
199.25
301
```

Rezultatas (išvestis, `cout`):
```
239.5
180
199.25
```

## 2 užduotis - Banko pavedimai

Duomenų faile `duomenys.txt` pirmasis skaičius nurodo, kiek pavedimų atlikta į banko sąskaitą. Toliau pateikti pavedimų dydžiai: neigiamas skaičius reiškia išlaidas, teigiamas - pajamas. Atspausdinkite tik pajamas (teigiamus pavedimus).

**Reikalavimai**:
- Spausdinkite tik teigiamus pavedimus.
- Kiekvieną įnašą spausdinkite atskiroje eilutėje.

Įvestis (`duomenys.txt`):
```
7
-12.5
200
0
15.75
-80
3
-4
```

Rezultatas (išvestis, `cout`):
```
200
15.75
3
```
