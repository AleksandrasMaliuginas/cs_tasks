# Using Variables

## Alice in Paris
Given the following code template:

```cpp
#include <iostream>
using namespace std;

int main() {
    string name = "Alice";
    int age = 20;
    string city = "Paris";

    // TODO: use cout to print the sentence

    return 0;
}
```

Your task: Print exactly this sentence using the variables:
    
    My name is Alice, I am 20 years old and I live in Paris.
